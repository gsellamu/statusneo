How to read this: the center stack is the seven technical layers; the four pillars (top/left/bottom/right) are cross-cutting concerns that span all of them. Blue = Modutecture’s existing platform · green = our working prototype · gold = our proposed delivery · purple = proven elsewhere. Domain-first: we build the Kaiser-Permanente hospital vertical (layer 2) end-to-end first and extract the domain-agnostic foundation (layer 3) from it — one real vertical earns the platform. The invariant across every layer and pillar: the brain proposes, the deterministic gate disposes, the twin remembers.
Framework alignment: this architecture maps to the production-grade enterprise agentic-AI pattern — seven agent-system layers (interface · orchestration · runtime · memory · tools · observability · governance) on four pillars (bounded autonomy · contextual grounding · orchestration · governance/explainability). Independently convergent with the current literature: Kellton — Enterprise Agentic AI Architecture (2026); Fareed Khan / Asimsultan / JIN — The 7 Layers of a Production-Grade Agentic AI System (2025–26); AIMultiple — Agentic AI Stack moat analysis. Where we deliberately specialize: a deterministic compliance gate — not a probabilistic governance agent — for hospital code-safety, and domain-first verticalization, building moat exactly where the literature says it accrues: cognition, governance, and domain specialization.
