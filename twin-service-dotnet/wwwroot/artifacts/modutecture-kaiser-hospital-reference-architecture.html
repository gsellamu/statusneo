BOne declared source of truth

Geometry is not smeared across three stores — it is owned by one.

TRUTH — the twin (L7)
The event-sourced parametric definition + knowledge graph is authoritative. It is the thing that gets mutated. Every geometry change is an event-sourced commit here.
PROJECTION — Unity (L1)
Unity renders / re-triangulates from the twin's procedural definition. The thin-client re-triangulation pattern is correct only because the definition is fed from the twin.
FEDERATION — Revit / IFC
Optional export / interop only. Never the geometry engine in the live loop. Modutecture is 100% Unity-native; Revit/IFC federates out, it does not own truth.

Mini-LLM is not the compliance authority. Fine-tuning a 7B on FGI/ADA buys jargon fluency and token efficiency — nothing more. Codes are enforced by the deterministic gate and grounded by RAG over the actual code text; weights produce confident, wrong citations. Language ≠ compliance.

CWorked example — MRI relocation (twin as truth, no Revit in loop)

"Move MRI Room 2 m north toward the structural column." The authoritative geometry change happens at the twin commit (step 6) — not in Unity, not in Revit.

Input guardrail — sanitize, check authorization (graduated: code-affecting → requires approval).

Orchestrator — open the LangGraph state graph; update HospitalState.
4
L4 · PODS
Agent pods — Structural runs OpenSees on load near the column; MEP queries the graph → finds the chilled-water-line clash (blast-radius).
5
L2 · GATE
Gate (output) — validate proposed position vs FGI clearances + structural limits; on fail → halt + force recalibration.
6
L7 · TWIN
Twin commit — the parametric definition of MRI-Room is updated; this is the authoritative geometry change, event-sourced.
7
L1 · UNITY
Unity sync — twin emits the delta → Unity re-renders from the new definition.

Revit appears nowhere in this loop. It is invoked only if separately exporting to BIM — federation, not the geometry engine. The source doc's "Revit shifts the geometry nodes" step is removed by design.
