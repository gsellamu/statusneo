2The 80:20 operating model

“FDE” is now a two-part entity. The agentic swarm owns velocity, breadth, and repetition; the human Pod Commander owns the four things an agent cannot be accountable for — and is Accountable for every stage.

The Agentic FDE

The 80% · machine-owned
  • Spec Architect — intent & spec → machine-executable plan
  • Systems Engineer — writes the code, builds the binary
  • Adversarial SDET — tests & stress-tests for full coverage
  • Security Sentinel — SAST, dependency & compliance scans
  • DevOps Overseer — zero-trust deploy within guardrails
  • Reliability Agent — monitors, maps traces, auto-heals

The Human Pod Commander

The 20% · judgment-owned
  • Diagnosis & architecture — reading the client's real problem
  • Client trust & evangelism — a CISO trusts a person, not a bot
  • Spec authorship & intent — a wrong spec fails at machine speed
  • Governance & the ship gate — the irreversible call
  • Exception triage — breaks the tie when the swarm loops
