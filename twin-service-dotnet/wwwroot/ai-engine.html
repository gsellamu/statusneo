The honest framing for the room: this is a governed agentic stack, not a list of logos. We chose LangGraph for an auditable state machine with human checkpoints; AutoGen and CrewAI are evaluated alternatives for multi-agent patterns, swappable behind MCP. Retrieval is hybrid — a GraphDB (Neo4j) for relational codes plus a VectorDB (Qdrant/pgvector) for the code corpus. The model is local-first today, with a custom domain-tuned LLM as the moat on the roadmap. And we plug into the AEC industry's AI — Autodesk Forma, Scan-to-BIM, generative design — rather than reinventing it. Through all of it, the invariant holds: the LLM proposes, the deterministic gate disposes. That is what makes agentic AI safe enough for a hospital.
